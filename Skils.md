# Pisces Project Skills & Terminology

물고기자리 프리미엄 랜딩 페이지 개발 과정에서 활용된 핵심 기술 스택 및 전문 용어 정리입니다.

## 核心 (Core) 기술 스택

### 1. 전역 애니메이션 & 인터랙션
- **GSAP (GreenSock Animation Platform)**: 웹 브라우저에서 고성능, 정교한 애니메이션을 구현하기 위한 업계 표준 자바스크립트 라이브러리입니다.
- **GSAP ScrollTrigger**: 스크롤 위치에 따라 요소의 애니메이션을 트리거하거나, 진행 상태를 제어하는 플러그인입니다.
- **Lenis (Smooth Scroll)**: 브라우저 기본 스크롤에 관성을 부여하여 고급스럽고 부드러운 스크롤 경험을 제공하는 라이브러리입니다.
- **Stagger Animation**: 여러 요소가 순차적으로(시차를 두고) 나타나게 하여 시각적 리듬감을 주는 애니메이션 기법입니다.

### 2. UI/UX 디자인 패턴
- **Split Sticky Layout**: 화면을 수직으로 나누어 한쪽은 고정(Sticky)하고 다른 한쪽은 스크롤되게 하여 시각적 묵직함과 스토리텔링을 동시에 잡는 레이아웃입니다.
- **Mega Menu (Full-width)**: 개별 드롭다운 대신 화면 전체 너비를 활용하는 거대한 내비게이션 시스템으로, 정보량이 많을 때 '꽉 찬' 브랜드 이미지를 전달하기 적합합니다.
- **Glassmorphism**: 배경을 흐릿하게(Backdrop Blur) 처리하고 투명도를 조절하여 유리 같은 질감을 주는 현대적인 디자인 트렌드입니다.
- **Cross-fade Background**: 요소 위에 호버하거나 스크롤할 때 배경 이미지가 부드럽게 교체되는 연출 기법입니다.

### 3. 개발 프레임워크 & 아키텍처
- **EJS (Embedded JavaScript)**: 자바스크립트 기반의 템플릿 엔진으로, HTML 내부에 선언적 로직을 포함하여 컴포넌트화된 웹 페이지 작성을 가능케 합니다.
- **Semantic HTML5**: 브라우저와 검색 엔진이 웹 페이지의 구조와 의미를 명확히 이해할 수 있도록 설계된 마크업 체계입니다.
- **CSS Variable (Tokens)**: 컬러, 트랜지션, 폰트 크기 등을 변수화하여 프로젝트 전반의 일관성을 유지하고 유지보수를 용이하게 합니다.

## 전문 용어 (Glossary)

- **Slogan Parallax**: 슬로건 텍스트와 배경 요소가 서로 다른 속도로 움직여 입체감을 주는 효과입니다.
- **Viewport-based Entrance**: 요소가 화면에 나타나는 시점(Viewport)을 감지하여 애니메이션을 실행하는 방식입니다.
- **Z-Index Layering**: 요소들의 수직 중첩 순서를 관리하여 메가 메뉴나 스티키 레이어가 의도한 대로 겹치거나 가려지게 조절하는 기법입니다.
- **Tone & Manner Consistency**: 브랜드의 상징색(Navy, Gold)과 분위기를 디자인 전반에 일관되게 적용하여 브랜드 정체성을 확립하는 과정입니다.
